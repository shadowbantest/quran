# Al-Quran - Read, Listen & Study the Holy Quran

A modern, full-featured Quran web application built with React, TypeScript, and Tailwind CSS.

## Features

- **114 Surahs** with Arabic text (Uthmani script) and metadata
- **20+ Translations** including Sahih International, Yusuf Ali, Pickthall, and more
- **8 Reciters** with verse-by-verse audio (Alafasy, Abdul Basit, Al-Sudais, etc.)
- **Full-text Search** across translations with instant surah suggestions
- **Bookmarks** with local storage persistence
- **3 Themes** - Light, Dark, and Sepia
- **Adjustable Font Sizes** for Arabic and translation independently
- **Juz/Para Navigation** - browse all 30 parts
- **Sajda Markers** on prostration verses
- **Copy & Share** individual verses
- **Last Read Tracking** - continue where you left off
- **Responsive Design** - works on desktop and mobile

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- React Router v6
- [AlQuran.cloud API](https://alquran.cloud/api)
- Amiri Arabic font

Quran text and translations powered by [AlQuran.cloud](https://alquran.cloud). Original Quran text sourced from [Tanzil.net](https://tanzil.net).
